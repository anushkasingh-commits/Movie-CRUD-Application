# 🎬 Movie CRUD Application

A simple Movie Collection Management System built using Node.js, Express.js, MongoDB, and EJS.

## Features

- Add new movies
- View all movies
- Search movies by name
- Edit movie details
- Delete movies

## Technologies Used

- Node.js
- Express.js
- MongoDB
- Mongoose
- EJS
- HTML
- CSS

## Project Structure

Movie-CRUD-Application
├── config
├── controllers
├── models
├── routes
├── views
├── app.js
├── package.json
└── README.md

## Run the Project

1. Install dependencies

```bash
npm install
```

2. Start the server

```bash
nodemon app.js
```

or

```bash
node app.js
```

3. Open in browser

```text
http://localhost:4000
```
